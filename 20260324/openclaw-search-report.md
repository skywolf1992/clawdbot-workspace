# Openclaw 搜索简报

**日期：2026-03-24**

## 概述

OpenClaw 是一个开源的个人AI助手，能够在本地运行并执行各种任务。它被称为"会做事的人工智能助手"，与传统的聊天机器人不同，OpenClaw 可以真正在你的机器上执行操作。

## 核心特性

### 1. 本地运行
- 完全运行在用户自己的硬件上（笔记本、家用服务器或VPS）
- 不依赖云端服务，保证数据隐私

### 2. 多平台支持
- 支持 WhatsApp、Telegram、Discord、Slack、Signal、iMessage 等
- 可以同时连接多个平台

### 3. 自主执行能力
- 发送邮件
- 浏览网页
- 读写文件
- 执行shell命令
- 管理日历
- 与外部API交互

### 4. 持久记忆
- 跨会话记住用户偏好
- 保持对话上下文

### 5. 技能系统
- 可以通过编写技能(SKILL.md)扩展功能
- 自动化工作流程

## 发展历程

- **Warelay** → **Clawdbot** → **Moltbot** → **OpenClaw**
- 由 PSPDFKit 创始人 Peter Steinberger 创建
- GitHub 上获得 68,000+ 星标

## 争议与风险

1. 安全问题：WIRED报道有用户遭遇AI伪装成主人进行诈骗
2. 权限过大：可以访问和执行系统级操作
3. 建议：在隔离环境中使用，避免赋予过多权限
